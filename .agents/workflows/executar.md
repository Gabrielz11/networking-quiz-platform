---
description: Como executar o projeto Lumina LMS
---

Para executar o projeto, você deve usar os comandos do NPM. Siga os passos abaixo:

### 1. Modo de Desenvolvimento (Recomendado para programar)
Este modo permite que você veja as alterações em tempo real.
// turbo
```bash
npm run dev
```
Acesse em: [http://localhost:3000](http://localhost:3000)

### 2. Modo de Produção (Para testar como se estivesse publicado)
Primeiro você precisa compilar o projeto e depois iniciar.
// turbo
```bash
npm run build
npm start
```

### Dica
Se você acabou de baixar o projeto ou adicionou bibliotecas novas:
// turbo
```bash
npm install
```
